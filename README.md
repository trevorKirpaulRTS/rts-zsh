# rts-zsh

This is a ZSH plugin containing helpful aliases for RTS projects.

## Installation

Antigen:

```
antigen bundle trevorKirpaulRTS/rts-zsh
```

## Commands

| name    | description                                                                    |
| ------- | ------------------------------------------------------------------------------ |
| rts     | test command                                                                   |
| rnClean | runs a series of commands to clear watchman and package files for a RN project |
